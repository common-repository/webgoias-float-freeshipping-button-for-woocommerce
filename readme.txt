Tenha um botão flutuante para avisar o valor que falta para ganhar frete grátis

== Description ==
Adicione a seu Woocommerce um botão flutuante personalizável que verifica o valor total do carrinho e calcula o quanto falta para atingir o valor mínimo para ganhar frete grátis.

A forma de entrega "Frete Grátis" deve estar criada previamente nos métodos de entrega.

Nesta versão, não é possível configurar fretes diferentes para cada zona de entrega.

== Installation ==
1. Faça o upload do plugin através do painel do WordPress, indo em "Plugins" > "Adicionar Novo" ou Instale pelo Repositório WordPress
2. Ative o plugin indo até o menu "Plugins";
3. Configure suas opções em "Float Freeshipping Button".
